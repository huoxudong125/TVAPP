## 📢 下载与安装须知（设备版本选择指南）
由于 Android 电视与机顶盒设备硬件差异巨大，为了大家获得更好的性能表现并尽量减少安装包体积，我们提供了针对不同硬件平台的分包下载。

一般情况下，如果不清楚自己设备的架构：请直接优先下载带有 arm32、armeabi-v7a 或 universal（全量包）字样的版本，兼容性最高！

以下是各个包的具体说明，请对照自身设备进行下载：

### 📺 1. 架构篇（CPU 位数怎么选？）
  arm32 / armeabi-v7a / armeabi（⭐ 最推荐 / 兼容95%以上电视）
    适用设备：绝大多数的智能电视（海信、创维、小米等）、传统电视盒子（如小米盒子、天猫魔盒、当贝盒子标准版等）、智能投影仪、车机系统。
    特点：兼容性极强。不仅能在 32 位设备跑，在绝大部分 64 位的电视系统上也能向下兼容运行。如果不确定，下这个准没错！

  arm64 / arm64-v8a（🚀 性能版）
    适用设备：只支持 64 位架构的新款高端设备（如配置较好的近两年新款旗舰电视、高端外贸盒子、以及您如果想装在现代大部分 Android 智能手机上测试）。
    特点：运行效率更高，内存寻址更大，但如果装在老旧 32 位电视上会提示“无法解析该安装包”或“安装失败”。
